# The Workshop Factory Landing Page

A modern, responsive single-page marketing site built with **Bootstrap 5.3**, **AOS (Animate On Scroll)**, and lightweight custom CSS/JS.

## Tech stack

- HTML5 + semantic sections
- Bootstrap 5.3.3 (layout, components, scrollspy, validation classes)
- AOS 2.3.1 (scroll animations)
- Font Awesome icons
- Vanilla JavaScript for form validation and dynamic footer year

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Key improvements

- Added a stronger hero section with clearer headline and CTA styling.
- Introduced AOS animation framework for progressive entrance animations.
- Added skip-link and Bootstrap Scrollspy for accessibility and navigation feedback.
- Upgraded contact form UX with required fields and client-side Bootstrap validation states.
- Improved service cards, testimonials, and section spacing/contrast for cleaner visual hierarchy.
- Added lazy loading on non-critical images for faster initial render.
