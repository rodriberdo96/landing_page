# The Workshop Factory Landing Page

Simple single-page marketing website built with HTML, CSS, Bootstrap 5, and Font Awesome.

## Quick start

Open `index.html` directly in your browser, or serve the project locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Improvements included

- Improved semantic structure (`header`, `main`, `section`, `footer`) for better accessibility.
- Fixed layout and responsiveness issues in mobile styles.
- Corrected broken class names and typoed container class.
- Removed leading slashes from local image paths to make the page portable across hosting environments.
- Improved visual contrast in dark overlay sections for readability.
